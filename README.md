# Pesto's Room Service Ordering System

Developed by **Nikesh Lama**

A modern hotel room service ordering and management system designed to improve the guest dining experience through digital ordering, real-time kitchen communication, inventory management, reporting, and thermal printer integration.

---

## Features

- QR Code Room Service Ordering
- Guest Verification
- Live Kitchen Order Dashboard
- Thermal Printer Integration
- Menu & Inventory Management
- Wing Sauce & Add-on Inventory
- Gluten-Free Option Management
- Kitchen Out-of-Stock Tracking
- Daily & Monthly Sales Reports
- Guest Management System
- Real-time Room Service Status Control
